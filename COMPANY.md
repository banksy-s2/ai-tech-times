# AI TECH TIMES 編集部(株)

総合ニュースサイト「AI TECH TIMES」を運営するAI会社。
カテゴリは **AI / シリコンバレー最速(英語ソースを日本語翻訳) / 海外AIの声(識者個人の発信を翻訳紹介) / インフルエンサー / 時事・世界** の5つ + **世界のバズ動画TOP10ランキング**。
記事の収集・選定・執筆・公開・X告知まで、1日4回(朝7:00・昼12:30・夕方17:30・夜21:30 JST)全自動で回る。

- サイト: https://ai-tech-times.web.app/
- 運用コスト: 月$0(Gemini無料枠 + GitHub Pages + GitHub Actions無料枠)
- 設立: 2026-07-21

## 業務フロー(1日4回: 朝7:00・昼12:30・夕方17:30・夜21:30 JST)

1. **久遠**: カテゴリ別にRSSを巡回(AI 9ソース / シリコンバレー: Techmeme・TechCrunch・The Verge・Ars・CNBC / 海外AIの声: Willison・Mollick・Clark・Lambert・Zvi・Karpathy・Raschkaの個人ブログ / インフルエンサー: Googleニュース / 時事・世界: NHK+BBC)、既報を除外
2. **真行寺**: Gemini(gemini-flash-latest)でカテゴリごとに選定(1回あたりAI2・シリコンバレー2・海外AIの声1・インフルエンサー1・時事2=計32本/日)。英語ソースは日本語翻訳して執筆
3. **真行寺**: 各ネタを500〜800字の日本語記事に執筆(出典明記、捏造禁止)
4. **久遠**: YouTube急上昇(米・英・日・韓・伯・印)を統合し世界バズ動画TOP10を集計、真行寺が一言コメント
5. **八重樫**: 静的サイトをdocs/に再生成(カテゴリ別ページ/buzz.html/JSON-LD/OGP/sitemap/llms.txt/RSS込み)
6. **桐生**: 新着記事をXに告知投稿(トークン未設定時はスキップ)
7. GitHub Actionsがdata/とdocs/をcommit → GitHub Pagesが即時公開

## 編集方針

- 一次情報(元記事)のタイトル・要約に書かれた事実だけを使う。推測で数値や固有名詞を足さない
- 必ず出典名とURLを記事末尾に明記する
- 同じ話題は1日1本まで。日本語圏の読者向けに、海外ネタは背景を1文補う
