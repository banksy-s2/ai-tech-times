AMD、AI向け新システム「Helios」をAzureに大規模導入

AI2026-07-21 19:01 / AMDMicrosoftAIAzureHelios
AMDはMicrosoftとの戦略的パートナーシップを拡大し、AIデータセンター向けラックスケールシステム「AMD Helios」をMicrosoft Azureへ導入すると発表しました。

現地時間の2026年7月20日、半導体大手のAMDは、Microsoftが提供するクラウドコンピューティングサービス「Microsoft Azure」との戦略的パートナーシップを大幅に拡大することを発表しました。今回の提携強化は、AI開発の加速を見据えたもので、AMDの技術基盤をAzureのインフラに深く統合する狙いがあります。

このパートナーシップの核となるのが、AMDが新たに展開するラックスケールAIシステム「AMD Helios」の導入です。Heliosは、AMD製のCPU、GPU、ネットワーク機器、そして最適化されたソフトウェアを網羅した包括的なシステムであり、AIデータセンターの処理能力を飛躍的に高めることを目的としています。AMDは、このシステムをMicrosoft Azure上で大規模に展開することで、急増するAIワークロードの需要に応える計画です。

AMDは、高性能な半導体製品を設計・開発する世界的なメーカーであり、特にGPU分野ではNVIDIAの強力な競合として知られています。今回の「AMD Helios」の大規模導入により、クラウド環境におけるAIインフラの選択肢が広がり、業界内での競争がさらに激化することが予想されます。
