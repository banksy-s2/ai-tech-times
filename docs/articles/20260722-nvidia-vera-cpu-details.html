NVIDIAの新CPU「Vera」公開、他社製比2.2倍の性能

日本のAI2026-07-22 21:02 / NVIDIACPUAI
⚡ 3行まとめ
  • NVIDIAがAI向けCPU「Vera」詳細公開
  • 独自コア88基搭載でSapphire Rapids比2.2倍
  • 2026年秋からシステムメーカー等へ提供予定
NVIDIAがAIエージェントやAIデータセンター向けCPU「NVIDIA Vera CPU」の詳細な設計を公開しました。

「AI TECH TIMES」編集部がお伝えします。NVIDIAは、AIエージェントやAIデータセンター向けに開発したCPU「NVIDIA Vera CPU」の詳しい設計を公開しました。

Veraは、NVIDIAが独自設計したデータセンター向けCPUコア「Olympus」を88基搭載しており、合計176スレッドを処理することが可能です。すでに量産段階に入っており、DeepInfraやCoreWeaveによる実機のベンチマーク結果も公開されています。その中でDeepInfraの実測では、Sapphire Rapids比で2.2倍の性能を示しました。

本製品は、2026年秋からシステムメーカーやクラウド事業者を通じて提供される予定です。今後の展開に大きな注目が集まっています。
