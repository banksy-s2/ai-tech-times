NVIDIA、ロボット用AI「Cosmos 3 Edge」公開

AI2026-07-21 / NVIDIACosmos 3 Edgeロボット自動運転世界モデル
半導体大手のNVIDIAは、ロボットや自動運転車向けの世界モデル「Cosmos 3 Edge」を発表しました。単一のグラフィックボードで動作する、ローカル向けの小型モデルです。

半導体大手のNVIDIAは、ロボットや自動運転車などの自律システム向けに設計された新しい世界モデル「Cosmos 3 Edge」を発表しました。このモデルは、外部のクラウドサーバーに依存することなく、ローカル環境で直接動作させることができる点が大きな特徴となっています。

発表された「Cosmos 3 Edge」は、40億パラメーターという比較的小型な規模に抑えられたAIモデルです。そのため、大規模なサーバー群を必要とせず、単一のグラフィックボードを搭載したシステムだけで十分に動作させることが可能であるとのことです。これにより、ロボットや自動運転車が現場でリアルタイムに判断を下すエッジAIとしての活用が期待されます。

開発元であるNVIDIAは、AI向けGPUの提供や自動運転技術、ロボット開発プラットフォームの構築において世界をリードする企業として知られています。今回公開された「Cosmos 3 Edge」は、限られたハードウェア資源でも高度なAI処理を実現するものであり、今後のロボット技術や自動運転分野の発展に大きく貢献するものとみられます。
